# Stock Management - Flask + MongoEngine

Requirements:
- Python 3.9+
- MongoDB (local or remote)
- pip install -r requirements.txt

1. Copy `.env.example` to `.env` and update MONGODB_URI.
2. Install:
   pip install -r requirements.txt
3. Run:
   python run.py

API:
- POST /product
- GET /product
- GET /product/<id>
- PUT /product/<id>
